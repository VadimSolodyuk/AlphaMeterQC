# Code Review

Проведи code review для следующего кода:

{{code}}

## Критерии проверки
1. **Безопасность**: Есть ли уязвимости?
2. **Производительность**: Есть ли узкие места?
3. **Читаемость**: Понятен ли код?
4. **Тесты**: Покрыт ли код тестами?
5. **Best practices**: Следует ли код стандартам проекта?

## Формат ответа
- ✅ Хорошо: [что сделано правильно]
- ⚠️ Внимание: [что можно улучшить]
- ❌ Критично: [что нужно исправить обязательно]

Предложи конкретные исправления для каждого замечания.
