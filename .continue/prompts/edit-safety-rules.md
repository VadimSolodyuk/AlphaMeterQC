## `edit_existing_file` — ЗАПРЕЩЁН
Полностью перезаписывает файл. **Не используй никогда**. Для создания файлов — только `create_new_file`.

## Только `single_find_and_replace`
Для любых изменений в существующих файлах.  
**Обязательно:**  
- 3–5 строк контекста (скопируй через `read_file`).  
- Если замена не уникальна — расширь контекст.

## Последовательность
Один edit-вызов за раз. Параллельные вызовы запрещены.

## Не более 3 файлов за раунд
Больше — разбей на раунды с подтверждением.

## При ошибке `single_find_and_replace`
1. Перечитай файл (`read_file`).  
2. Возьми ≥5 строк контекста.  
3. Повтори.
