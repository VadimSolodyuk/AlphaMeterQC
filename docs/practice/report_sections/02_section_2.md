# 2. Разработка концептуальной и логической модели базы данных

В рамках второго этапа работ на основе результатов анализа требований и определённых ключевых сущностей была разработана концептуальная и логическая модель базы данных. Детальное описание выполненных работ по каждому из подпунктов приведено в соответствующих разделах:
а) **пункт 2.1** — создание ER-диаграммы (модель «сущность-связь») (см. файл `02_subsection_1.md`);
б) **пункт 2.2** — проектирование логической модели базы данных (см. файл `02_subsection_2.md`);
в) **пункт 2.3** — проведение нормализации данных (см. файл `02_subsection_3.md`).
