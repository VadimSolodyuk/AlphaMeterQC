В рамках четвёртого этапа работ были выполнены оптимизация запросов, нагрузочное тестирование и анализ узких мест в производительности созданной базы данных. Детальное описание выполненных работ по каждому из подпунктов приведено в соответствующих разделах:

б) **пункт 4.2** — проведение нагрузочного тестирования (см. файл `04_subsection_2.md`);
в) **пункт 4.3** — анализ и устранение узких мест в производительности (см. файл `04_subsection_3.md`).
