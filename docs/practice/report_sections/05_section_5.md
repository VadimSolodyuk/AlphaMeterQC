б) **пункт 5.2** — обеспечение безопасности данных: контроль доступа и шифрование (см. файл `05_subsection_2.md`);
в) **пункт 5.3** — подготовка отчетной документации по выполненным работам (см. файл `05_subsection_3.md`).
