# 3. Реализация базы данных в СУБД

В рамках третьего этапа работ на основе разработанной логической модели была выполнена физическая реализация базы данных в выбранной СУБД. Детальное описание выполненных работ по каждому из подпунктов приведено в соответствующих разделах:

а) **пункт 3.1** — создание базы данных и таблиц в выбранной СУБД (см. файл `03_subsection_1.md`);
б) **пункт 3.2** — определение связей и ограничений (первичные и внешние ключи) (см. файл `03_subsection_2.md`);
в) **пункт 3.3** — разработка SQL-скриптов для создания и управления объектами базы данных (см. файл `03_subsection_3.md`).
